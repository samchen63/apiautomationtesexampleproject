1. This API automation test project uses JSON as an example.
2. This API automation test project relies on self-hosted JSON API provider
3. To set up self-hosted JSON API provider, please follow the instructions
	a. Install Node.js (https://nodejs.org/en/)
	b. Unzip restful-booker-master.zip to a local folder e.g restful-booker-master
	c. Navigate to restful-booker-master folder
	d. Open a command prompt
	e. Run "npm install" and wait for installation to be completed
	f. Run "npm start" and keep command prompt opened
	
Note: This self-hosted JSON API provider uses port number 3001, please ensure that port 3001 is free and unblocked
